# Theorem 2.12 (Matrix mult properties)
Let $A$ be an $m\times n$ matrix, $B,C$ be $n\times p$ matrices, $D,E$ be $q\times m$ matrices.
Then
1. $A(B+C)=AB+AC$ and $(D+E)A=DA+EA$
2. $a(AB)=(aA)B=A(aB)$ for all $a\in F$.
3. $AI_{n}=I_{m}A=A$
4. If $V$ is an $n$-dimensional vector space with an ordered basis $\beta$, then $[I_{V}]_{\beta}=I_{n}$
