# Definition (Local classifications of a surface)
Let $S$ be an oriented surface with orientation $N$.
Let $p\in S$.
We call $p$ 
1. Elliptic if $K>0$
2. Hyperbolic if $K<0$
3. Parabolic if $K=0,dN_{p}\neq 0$
4. Planar if $K=0,dN_{p}=0$
