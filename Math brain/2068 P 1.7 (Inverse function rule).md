# Proposition 1.7 (Inverse function rule)
Let $I=(a,b)\in\power(\mathbb{R})$, $c\in I$ and $f:I\to \mathbb{R}$ be a function.
Denote $d=f(c)$
Suppose:
1. $f$ is strictly increasing 
2. $J=f(I)$ is an interval.
3. $f'(c)$ exists
4. $f'(c)\neq 0$

Then $g'(d)$ exists and $g'(d)=\dfrac{1}{f'(c)}$
